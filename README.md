# Seed

A flexible and scalable suite for building modern websites, using nodejs with express.

## Getting started

Install nodejs packages

```
$ npm install
```

Install front-end packages

```
$ bower install
```

Run as debug mode

```
$ gulp debug
```

Or as production mode

```
$ gulp start
```
